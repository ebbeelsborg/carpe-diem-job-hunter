### **Key User Flows:**

**1. Add Application Flow:**
```
Dashboard → "Add Application" button → Modal opens
→ Fill company info → Save → Card appears in list
→ Toast: "Application added successfully!"
```

**2. Schedule Interview Flow:**
```
Application card → "Schedule Interview" button → Modal opens
→ Fill interview details → Save → Interview appears in timeline
→ Toast: "Interview scheduled for [date]"
```

**3. Update Status Flow:**
```
Application card → Status dropdown → Select new status
→ Auto-save → Card color updates → Stats refresh
→ Toast: "Status updated to [new status]"
```

**4. Post-Interview Flow:**
```
Dashboard → Upcoming interview → "Complete" button → Modal
→ Add notes, rating, questions → Save
→ Status auto-updates to "Completed" → Stats refresh
```

**5. Resource Discovery Flow:**
```
Resources tab → "Add Resource" button → Modal
→ Add title, URL, category → Optionally link to company
→ Save → Resource card appears → Toast: "Resource saved"
```

**Frontend:**
- React 18 with Hooks
- TailwindCSS for styling
- shadcn/ui components (Dialog, Select, Badge, Card)

- date-fns or day.js for dates

**Backend:**
- Supabase:
  - PostgreSQL database

  - Supabase Auth (email/password)
  - Real-time subscriptions (optional - for live updates)
  - REST API auto-generated

**Hosting:**
- Netlify (with environment variables for Supabase keys)

**State Management:**
- React Context or Zustand for:

  - Filter preferences
